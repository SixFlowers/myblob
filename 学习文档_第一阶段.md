# 第一阶段：最小可行版 - 简单文件下载器

## 目标

创建一个最简单程序，实现：建立 TCP 连接 → 发送 HTTP 请求 → 打印服务器响应。

**代码量**：~50行

---

## 核心概念

### 1. 什么是 Socket？

**Socket（套接字）** 是网络通信的端点。

**类比**：就像打电话时的"听筒"
- IP 地址 = 电话号码（找哪台电脑）
- 端口号 = 分机号（找电脑上的哪个程序）
- Socket = 通话连接

### 2. TCP 连接过程

```
你的电脑                          服务器
   |                               |
   |  1. 发起连接请求              |
   |----------------------------> |
   |                               |
   |  2. 确认连接                  |
   |<---------------------------- |
   |                               |
   |  双向通信开始                 |
   |<============================>|
   |                               |
```

### 3. HTTP 请求格式

```
GET /index.html HTTP/1.1      <-- 请求行（方法 路径 协议版本）
Host: www.example.com          <-- 请求头
Connection: close              <-- 请求头

                               <-- 空行（分隔头部和正文）
```

### 4. HTTP 响应格式

```
HTTP/1.1 200 OK                <-- 状态行（协议 版本 状态码 状态消息）
Content-Length: 1234           <-- 响应头

                               <-- 空行
<html>这是网页内容</html>       <-- 响应正文
```

## 需要用到的系统函数

### 1. 创建 Socket
```cpp
#include <sys/socket.h>

int socket(int domain, int type, int protocol);
// 参数：
//   domain: AF_INET (IPv4)
//   type: SOCK_STREAM (TCP)
// 返回：socket 文件描述符（类似文件句柄）
```

### 2. 连接到服务器
```cpp
#include <sys/socket.h>

int connect(int sockfd, const struct sockaddr *addr, socklen_t addrlen);
// 参数：
//   sockfd: socket 描述符
//   addr: 服务器地址
//   addrlen: 地址长度
```

### 3. 发送数据
```cpp
#include <sys/socket.h>

ssize_t send(int sockfd, const void *buf, size_t len, int flags);
// 参数：
//   sockfd: socket 描述符
//   buf: 要发送的数据
//   len: 数据长度
// 返回：实际发送的字节数
```

### 4. 接收数据
```cpp
#include <sys/socket.h>

ssize_t recv(int sockfd, void *buf, size_t len, int flags);
// 参数：
//   sockfd: socket 描述符
//   buf: 接收缓冲区
//   len: 缓冲区大小
// 返回：实际接收的字节数（0表示连接关闭，-1表示错误）
```

### 5. 关闭连接
```cpp
#include <unistd.h>

int close(int fd);
// 参数：文件描述符
```

### 6. 域名解析
```cpp
#include <netdb.h>

struct hostent *gethostbyname(const char *name);
// 参数：域名（如 "www.example.com"）
// 返回：主机信息结构体（包含 IP 地址）
```

## 设计步骤

### 步骤 1：解析服务器地址

需要把域名（如 `www.example.com`）转换成 IP 地址。

**提示**：
- 使用 `gethostbyname()` 函数
- 返回的 `hostent` 结构包含 `h_addr_list`（IP 地址列表）
- IP 地址在 `h_addr` 中

**代码框架**：
```cpp
// 1. 解析域名获取 IP
struct hostent *server = gethostbyname("www.example.com");
if (server == nullptr) {
    // 错误处理
}

// 2. 填充地址结构
struct sockaddr_in addr;
addr.sin_family = AF_INET;  // IPv4
addr.sin_port = htons(80);  // HTTP 端口，转换为网络字节序
// 复制 IP 地址到结构体
memcpy(&addr.sin_addr.s_addr, server->h_addr, server->h_length);
```

### 步骤 2：创建 Socket

**提示**：
- 使用 `socket()` 函数
- 参数固定为：`AF_INET, SOCK_STREAM, 0`

**代码框架**：
```cpp
int sockfd = socket(AF_INET, SOCK_STREAM, 0);
if (sockfd < 0) {
    // 错误处理
}
```

### 步骤 3：建立连接

**提示**：
- 使用 `connect()` 函数
- 地址结构需要强制转换为 `struct sockaddr *`

**代码框架**：
```cpp
if (connect(sockfd, (struct sockaddr *)&addr, sizeof(addr)) < 0) {
    // 错误处理
}
```

### 步骤 4：发送 HTTP 请求

**提示**：
- 使用 `send()` 函数
- HTTP 请求是纯文本字符串
- 格式：`"GET /path HTTP/1.1\r\nHost: hostname\r\n\r\n"`

**代码框架**：
```cpp
const char *request = "GET / HTTP/1.1\r\nHost: www.example.com\r\n\r\n";
send(sockfd, request, strlen(request), 0);
```

### 步骤 5：接收响应

**提示**：
- 使用 `recv()` 函数
- 需要循环接收（一次可能收不完）
- `recv()` 返回 0 表示服务器关闭连接
- 将接收的数据存入缓冲区

**代码框架**：
```cpp
char buffer[4096];
while (1) {
    int n = recv(sockfd, buffer, sizeof(buffer) - 1, 0);
    if (n <= 0) break;  // 连接关闭或出错
    buffer[n] = '\0';  // 添加字符串结束符
    printf("%s", buffer);  // 打印响应
}
```

### 步骤 6：关闭连接

**提示**：
- 使用 `close()` 函数

**代码框架**：
```cpp
close(sockfd);
```

## 常见错误和解决

### 错误 1：域名解析失败
```
error: unknown host
```
**原因**：`gethostbyname()` 返回 `NULL`
**解决**：检查域名是否正确，检查网络连接

### 错误 2：连接被拒绝
```
Connection refused
```
**原因**：服务器没有运行或端口不对
**解决**：确认服务器地址和端口

### 错误 3：编译错误
```
undefined reference to `socket'
```
**原因**：缺少头文件
**解决**：确保 `#include <sys/socket.h>`
