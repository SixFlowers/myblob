## 第二版概述

### 什么是第二版？

第二版是第一版的**增强版**，主要解决第一版存在的几个严重问题：

| 问题 | 第一版 | 第二版 | 影响 |
|------|--------|--------|------|
| **错误处理** | 只有简单的 `cout` | 详细的 errno、strerror | 方便调试 |
| **超时机制** | 没有 | 5秒超时 | 防止程序卡死 |
| **URL 解析** | 有双斜杠bug | 智能路径拼接 | 下载失败 |
| **资源管理** | 可能泄漏 | Defer 自动释放 | 内存泄漏 |
| **HTTPS 支持** | 不完整 | 完整实现 | 无法访问 HTTPS |
| **日志系统** | 没有 | DEBUG/ERROR 区分 | 调试困难 |
| **多云架构** | 只支持 HTTP | Provider 架构 | 扩展性差 |

### 第二版只支持什么？

**HTTP 和 HTTPS**！

其他云服务商（AWS S3、MinIO、Azure、GCP 等）将在第三版及之后实现。

第二版主要是：
1. **打基础**：完善错误处理、超时、重试机制
2. **建架构**：设计 Provider 工厂模式（方便后续扩展）
3. **完善功能**：让 HTTP/HTTPS 下载稳定可靠

### 学习目标

完成第二版后，你应该理解：

1. **RAII 模式**：什么是资源获取即初始化，如何自动管理资源
2. **错误处理**：如何使用 errno、h_errno、strerror
3. **超时机制**：如何设置 socket 超时
4. **重试机制**：如何实现自动重试
5. **HTTPS**：如何在 TCP 之上添加 TLS 加密
6. **工厂模式**：如何设计可扩展的 Provider 架构

```cpp
// ✅ 改进：设置超时机制

// 1. 定义超时结构体
// struct timeval 有两个成员：
//   - tv_sec: 秒数（5 秒）
//   - tv_usec: 微秒数（0 微秒）
// 所以这个设置是 5 秒 + 0 微秒 = 5 秒
struct timeval timeout;
timeout.tv_sec = 5;    // 等待 5 秒
timeout.tv_usec = 0;   // 0 微秒（1 秒 = 1000000 微秒）

// 2. 设置发送超时
// SOL_SOCKET: 在 socket 层面设置选项
// SO_SNDTIMEO: 发送超时选项
// &timeout: 超时结构体的地址
// sizeof(timeout): 结构体大小
setsockopt(sockfd, SOL_SOCKET, SO_SNDTIMEO, &timeout, sizeof(timeout));

// 3. 设置接收超时
// SO_RCVTIMEO: 接收超时选项
// 和发送超时一样设置
setsockopt(sockfd, SOL_SOCKET, SO_RCVTIMEO, &timeout, sizeof(timeout));

// 现在，如果 5 秒内没有数据：
// - send() 会返回 -1，errno = EAGAIN 或 EWOULDBLOCK
// - recv() 会返回 -1，errno = EAGAIN 或 EWOULDBLOCK
```

### 功能 4：RAII 资源管理（Defer）

#### 什么是 RAII？

**RAII = Resource Acquisition Is Initialization**
**资源获取即初始化**

核心思想：
- 在**构造函数**中获取资源（打开文件、分配内存、创建 socket）
- 在**析构函数**中释放资源（关闭文件、释放内存、关闭 socket）
- C++ 自动调用析构函数，无论函数如何退出

#### Defer 类

> 📁 文件：`include/utils/defer.hpp`（**新建此文件**）

```cpp
// ============================================================================
// Defer 类 - RAII 通用清理器
// ============================================================================
// 功能：在对象离开作用域时执行自定义清理函数
// 原理：析构函数调用预先保存的清理函数
// 优点：比特定资源的 SocketGuard 更灵活
// ============================================================================

#pragma once
#include <functional>

namespace myblob::utils {

// ============================================================================
// Defer 类
// ============================================================================
// 
// 使用方法：
//   {
//       Defer d([]() {
//           // 清理代码
//       });
//       // ... 主逻辑 ...
//   } // d 的析构函数会执行清理代码
//
// 特点：
//   - 析构函数自动执行清理
//   - 支持提前释放（调用 release()）
//   - 支持任何清理逻辑
//
// ============================================================================

class Defer {
public:
    // ============================================================
    // 构造函数
    // ============================================================
    // 参数：cleanup - 清理函数
    // explicit 防止隐式转换
    // std::move 避免不必要的复制
    
    explicit Defer(std::function<void()> func) : func_(std::move(func)) {}
    
    // ============================================================
    // 析构函数 - 关键！
    // ============================================================
    // 如果没有提前调用 release()，析构函数会执行清理函数
    
    ~Defer() {
        if (func_) {  // 检查是否为空
            func_();  // 执行清理
        }
    }
    
    // ============================================================
    // release - 提前取消清理
    // ============================================================
    // 有时需要在析构函数之前提前清理
    // 调用 release() 后，析构函数不会再执行清理
    
    void release() {
        func_ = nullptr;  // 清空函数，不再执行
    }
    
    // ============================================================
    // 禁止拷贝和移动
    // ============================================================
    // Defer 对象不应该被拷贝或移动
    // 每个 Defer 应该绑定到一个资源
    
    Defer(const Defer&) = delete;
    Defer& operator=(const Defer&) = delete;
    Defer(Defer&&) = delete;
    Defer& operator=(Defer&&) = delete;

private:
    // 清理函数
    std::function<void()> func_;
};

}  // namespace myblob::utils
```

### 功能 5：重试机制

#### 为什么需要重试？

网络请求可能因为各种原因失败：
- 网络短暂中断
- 服务器过载（503 Service Unavailable）
- 限流（429 Too Many Requests）
- 临时故障

这些情况下，重试可能会成功。

#### 什么情况应该重试？

| 情况 | 是否重试 | 原因 |
|------|----------|------|
| 200-299 成功 | ❌ 不需要 | 已经成功了 |
| 400 Bad Request | ❌ 不重试 | 请求本身有问题，重试也没用 |
| 401 Unauthorized | ❌ 不重试 | 认证失败，重试也不会变 |
| 404 Not Found | ❌ 不重试 | 资源不存在，重试也不会变 |
| 429 Too Many Requests | ✅ 重试 | 服务器限流，等待后可能成功 |
| 500 Internal Server Error | ✅ 重试 | 服务器临时错误，可能恢复 |
| 502 Bad Gateway | ✅ 重试 | 网关错误，可能恢复 |
| 503 Service Unavailable | ✅ 重试 | 服务暂时不可用，可能恢复 |
| 网络超时 | ✅ 重试 | 网络可能恢复了 |

#### 重试策略

> 📁 文件：`include/network/http_client.hpp`（在 HttpClient 类中添加）

```cpp
// ============================================================================
// downloadWithRetry - 带重试的下载
// ============================================================================
// 参数：
//   url - 要下载的 URL
//   offset - 下载起始位置（断点续传）
//   length - 下载长度（0 表示下载全部）
//   maxRetries - 最大重试次数（默认 3）
//   timeoutSeconds - 超时秒数（默认 5）
// 返回：下载结果
// ============================================================================

HttpResponse HttpClient::downloadWithRetry(
    const std::string& url,
    uint64_t offset = 0,
    uint64_t length = 0,
    int maxRetries = 3,        // 默认最多重试 3 次
    int timeoutSeconds = 5      // 默认超时 5 秒
) {
    // lastResponse 用于保存最后一次的响应
    // 如果所有重试都失败，返回这个作为最终结果
    HttpResponse lastResponse;
    
    // ============================================================
    // 重试循环
    // ============================================================
    // 尝试 maxRetries 次
    // 如果某次成功了，立即返回
    // 如果某次失败了，检查是否应该重试
    
    for (int attempt = 1; attempt <= maxRetries; attempt++) {
        // 打印当前是第几次尝试
        std::cerr << "[DEBUG] Download attempt " << attempt << "/" << maxRetries << std::endl;
        
        // 1. 执行下载
        HttpResponse response = download(url, offset, length);
        
        // 2. 检查是否成功
        if (response.success) {
            // ✅ 下载成功！
            std::cerr << "[DEBUG] Download succeeded on attempt " << attempt << std::endl;
            return response;  // 立即返回，不需要重试
        }
        
        // ❌ 下载失败，保存结果
        lastResponse = response;
        
        // ============================================================
        // 判断是否应该重试
        // ============================================================
        
        // 情况 1：服务器错误（5xx）
        if (response.status_code >= 500) {
            // 服务器可能临时过载，等待后重试
            // 打印警告信息
            std::cerr << "[WARN] Server error " << response.status_code 
                      << ", retrying in " << attempt << " seconds..." << std::endl;
            
            // 指数退避策略：
            // 第 1 次失败：等 1 秒
            // 第 2 次失败：等 2 秒
            // 第 3 次失败：等 3 秒
            // 这样避免对服务器造成更大压力
            std::this_thread::sleep_for(std::chrono::seconds(attempt));
        }
        
        // 情况 2：限流（429）
        else if (response.status_code == 429) {
            // 被限流了，需要等待更长时间
            std::cerr << "[WARN] Rate limited (429), waiting 5 seconds..." << std::endl;
            
            // 限流时等待固定 5 秒
            // 这是服务器的要求，不是客户端决定的
            std::this_thread::sleep_for(std::chrono::seconds(5));
        }
        
        // 情况 3：连接超时
        else if (response.status_code == 0 && 
                 response.error_message.find("timeout") != std::string::npos) {
            // 连接超时，网络可能恢复了
            std::cerr << "[WARN] Connection timeout, retrying..." << std::endl;
            
            // 等待 1 秒后重试
            std::this_thread::sleep_for(std::chrono::seconds(1));
        }
        
        // 情况 4：其他客户端错误（4xx）
        else {
            // 请求本身有问题
            // 重试也不会改变结果
            std::cerr << "[ERROR] Non-retryable error: " << response.error_message << std::endl;
            
            // 打印具体的状态码
            if (response.status_code > 0) {
                std::cerr << "[ERROR] HTTP status: " << response.status_code << std::endl;
            }
            
            // 立即退出，不重试
            break;
        }
    }
    
    // ============================================================
    // 所有重试都失败了
    // ============================================================
    
    std::cerr << "[ERROR] All " << maxRetries << " attempts failed" << std::endl;
    
    // 返回最后一次的结果
    // 包含错误信息，供调用者分析
    return lastResponse;
}
```

```cpp
enum class CloudService : uint8_t {
    HTTPS = 0,          // HTTPS 加密
    HTTP = 1,           // 普通 HTTP
    AWS = 2,            // Amazon S3
    Azure = 3,          // Microsoft Azure Blob
    GCP = 4,            // Google Cloud Storage
    Oracle = 5,         // Oracle Cloud Object Storage
    IBM = 6,            // IBM Cloud Object Storage

// RemoteInfo 结构体
// ============================================================================
// 功能：存储从 URL 解析出来的信息
// 这些信息用于创建对应的 Provider
// ============================================================================

struct RemoteInfo {
    /// 云服务商类型
    CloudService provider = CloudService::HTTPS;
    
    /// 存储桶名称
    std::string bucket = "";
    
    /// 区域
    std::string region = "";
    
    /// 端点地址
    std::string endpoint = "";
    
    /// 端口号
    uint32_t port = 80;
    
    /// 是否使用 HTTPS
    bool https = true;
};

// ============================================================================
// parseRemoteInfo - 解析 URL 获取 RemoteInfo
// ============================================================================
// 功能：从各种格式的 URL 中提取关键信息
// ============================================================================

inline RemoteInfo parseRemoteInfo(const std::string& url) {
    RemoteInfo info;  // 创建空的信息结构
    
    // ============================================================
    // 第一步：识别协议类型
    // ============================================================
    
    if (url.find("https://") == 0) {
        info.https = true;
        info.provider = CloudService::HTTPS;
        info.port = 443;
    }
    else if (url.find("http://") == 0) {
        info.https = false;
        info.provider = CloudService::HTTP;
        info.port = 80;
    }
    else if (url.find("s3://") == 0) {
        info.provider = CloudService::AWS;
        info.port = 443;
    }
    else if (url.find("azure://") == 0) {
        info.provider = CloudService::Azure;
        info.port = 443;
    }
    else if (url.find("gs://") == 0) {
        info.provider = CloudService::GCP;
        info.port = 443;
    }
    else if (url.find("minio://") == 0) {
        info.provider = CloudService::MinIO;
        info.port = 9000;
    }
    else if (url.find("oci://") == 0) {
        info.provider = CloudService::Oracle;
        info.port = 443;
    }
    else if (url.find("ibm://") == 0) {
        info.provider = CloudService::IBM;
        info.port = 443;
    }
    
    // ============================================================
    // 第二步：提取前缀长度
    // ============================================================
    
    size_t prefix_length = 0;
    switch (info.provider) {
        case CloudService::HTTP:
            prefix_length = info.https ? 8 : 7;
            break;
        case CloudService::HTTPS:
            prefix_length = 8;
            break;
        case CloudService::AWS:
        case CloudService::MinIO:
            prefix_length = 5;
            break;
        case CloudService::Azure:
            prefix_length = 9;
            break;
        case CloudService::GCP:
            prefix_length = 5;
            break;
        case CloudService::Oracle:
            prefix_length = 6;
            break;
        case CloudService::IBM:
            prefix_length = 6;
            break;
        default:
            prefix_length = 0;
            break;
    }
    
    // ============================================================
    // 第三步：提取主机部分
    // ============================================================
    
    std::string remaining = url.substr(prefix_length);
    size_t slash_pos = remaining.find('/');
    
    std::string host_part;
    if (slash_pos == std::string::npos) {
        host_part = remaining;
        remaining = "";
    } else {
        host_part = remaining.substr(0, slash_pos);
        remaining = remaining.substr(slash_pos + 1);
    }
    
    // ============================================================
    // 第四步：解析 Bucket:Region@Endpoint:Port
    // ============================================================
    // URL 格式：provider://bucket:region@endpoint:port/path
    // 例如：s3://my-bucket:us-east-1@s3.amazonaws.com:443/path/to/file.txt
    
    size_t at_pos = host_part.find('@');
    
    if (at_pos != std::string::npos) {
        // 有 @ 分隔符：bucket_region@endpoint_part
        std::string bucket_region = host_part.substr(0, at_pos);
        std::string endpoint_part = host_part.substr(at_pos + 1);
        
        // 解析 endpoint 和 port
        size_t colon_pos = endpoint_part.find(':');
        if (colon_pos != std::string::npos) {
            info.endpoint = endpoint_part.substr(0, colon_pos);
            info.port = static_cast<uint32_t>(std::stoi(endpoint_part.substr(colon_pos + 1)));
        } else {
            info.endpoint = endpoint_part;
        }
        
        // 解析 bucket 和 region
        size_t colon_pos2 = bucket_region.find(':');
        if (colon_pos2 != std::string::npos) {
            info.bucket = bucket_region.substr(0, colon_pos2);
            info.region = bucket_region.substr(colon_pos2 + 1);
        } else {
            info.bucket = bucket_region;
        }
    } else {
        // 没有 @ 分隔符：直接是 endpoint:port
        size_t colon_pos = host_part.find(':');
        if (colon_pos != std::string::npos) {
            info.endpoint = host_part.substr(0, colon_pos);
            info.port = static_cast<uint32_t>(std::stoi(host_part.substr(colon_pos + 1)));
        } else {
            info.endpoint = host_part;
        }
        
        // 尝试从 remaining 中提取 bucket（格式：bucket/path）
        if (!remaining.empty()) {
            size_t slash_in_remaining = remaining.find('/');
            if (slash_in_remaining != std::string::npos) {
                info.bucket = remaining.substr(0, slash_in_remaining);
            } else {
                info.bucket = remaining;
            }
        }
    }
    
    return info;
}
```

```cpp
        case CloudService::AWS:
        case CloudService::MinIO:
        case CloudService::Azure:
        case CloudService::GCP:
        case CloudService::Oracle:

        default: {
            std::cerr << "[ERROR] Unsupported cloud service: "
```

```cpp
std::string HttpClient::extract_host(const std::string& url) {

    if (start == std::string::npos) {
        start = 0;
    } else {
        start += 3;
    }
    
    size_t end = url.find("/", start);
    std::string host_port = url.substr(start, end - start);
    
    size_t colon = host_port.find(":");
    if (colon != std::string::npos) {
        return host_port.substr(0, colon);
    }
    
    return host_port;
}

// ============================================================================
// create_socket - 创建 TCP 连接
// ============================================================================

int HttpClient::create_socket(const std::string& host, uint16_t port) {
    std::cerr << "[DEBUG] create_socket: host='" << host 
              << "', port=" << port << std::endl;
    
    // 1. 创建 socket
    int sockfd = socket(AF_INET, SOCK_STREAM, 0);
    if (sockfd < 0) {
        std::cerr << "[ERROR] socket() failed: " << strerror(errno) << std::endl;
        return -1;
    }
    
    // 使用 Defer 管理 socket
    Defer defer([sockfd]() {
        std::cerr << "[DEBUG] Closing socket: fd=" << sockfd << std::endl;
        close(sockfd);
    });
    
    // 2. 设置超时（5秒）
    struct timeval timeout;
    timeout.tv_sec = 5;
    timeout.tv_usec = 0;
    setsockopt(sockfd, SOL_SOCKET, SO_SNDTIMEO, &timeout, sizeof(timeout));
    setsockopt(sockfd, SOL_SOCKET, SO_RCVTIMEO, &timeout, sizeof(timeout));
    
    // 3. 禁用 Nagle 算法
    int flag = 1;
    setsockopt(sockfd, IPPROTO_TCP, TCP_NODELAY, &flag, sizeof(flag));
    
    // 4. DNS 解析
    struct hostent* server = gethostbyname(host.c_str());
    if (server == nullptr) {
        std::cerr << "[ERROR] DNS lookup failed for: " << host << std::endl;
        std::cerr << "[ERROR] h_errno: " << h_errno << std::endl;
        
        switch (h_errno) {
            case HOST_NOT_FOUND:
                std::cerr << "[ERROR] Reason: Host not found" << std::endl;
                break;
            case TRY_AGAIN:
                std::cerr << "[ERROR] Reason: Try again later (DNS timeout)" << std::endl;
                break;
            case NO_RECOVERY:
                std::cerr << "[ERROR] Reason: Non-recoverable DNS error" << std::endl;
                break;
            case NO_DATA:
                std::cerr << "[ERROR] Reason: No address record" << std::endl;
                break;
            default:
                std::cerr << "[ERROR] Reason: Unknown DNS error" << std::endl;
                break;
        }
        
        return -1;
    }
    
    // 5. 设置服务器地址
    struct sockaddr_in addr;
    memset(&addr, 0, sizeof(addr));
    addr.sin_family = AF_INET;
    memcpy(&addr.sin_addr.s_addr, server->h_addr, server->h_length);
    addr.sin_port = htons(port);
    
    // 6. 建立连接
    if (connect(sockfd, (struct sockaddr*)&addr, sizeof(addr)) < 0) {
        std::cerr << "[ERROR] connect() failed: " << strerror(errno) << std::endl;
        std::cerr << "[ERROR] Could not connect to " << host << ":" << port << std::endl;
        return -1;
    }
    
    std::cerr << "[DEBUG] Socket created successfully: fd=" << sockfd << std::endl;
    
    // 成功！提前 release，避免 Defer 关闭 socket
    defer.release();
    
    return sockfd;
}

// ============================================================================
// send_request - 发送 HTTP 请求
// ============================================================================

HttpResponse HttpClient::send_request(int sockfd, const HttpRequest& request) {
    std::cerr << "[DEBUG] send_request: method=" << request.method 
              << ", path=" << request.path << std::endl;
    
    // 1. 构建 HTTP 请求
    std::string request_str = request.method + " " + request.path + " HTTP/1.1\r\n";
    request_str += "Host: " + request.host + "\r\n";
    request_str += "Connection: close\r\n";
    
    // 添加自定义头部
    for (const auto& header : request.headers) {
        request_str += header.first + ": " + header.second + "\r\n";
    }
    
    // 添加请求体
    if (!request.body.empty()) {
        request_str += "Content-Length: " + std::to_string(request.body.size()) + "\r\n";
    }
    
    // 结束头部
    request_str += "\r\n";
    
    // 添加请求体
    if (!request.body.empty()) {
        request_str += request.body;
    }
    
    // 2. 发送请求
    ssize_t sent = send(sockfd, request_str.c_str(), request_str.size(), 0);
    if (sent < 0) {
        std::cerr << "[ERROR] send() failed: " << strerror(errno) << std::endl;
        HttpResponse resp;
        resp.success = false;
        resp.error_message = "Failed to send request";
        return resp;
    }
    
    std::cerr << "[DEBUG] Sent " << sent << " bytes" << std::endl;
    
    // 3. 接收响应
    return recv_response(sockfd);
}

// ============================================================================
// recv_response - 接收 HTTP 响应
// ============================================================================

HttpResponse HttpClient::recv_response(int sockfd) {
    HttpResponse response;
    
    // ✅ 使用 std::vector<uint8_t> 代替 DataVector，方便索引
    std::vector<uint8_t> recv_buffer(8192);
    std::string header_str;  // 单独保存头部字符串
    bool headers_complete = false;
    size_t content_length = 0;
    size_t body_bytes_received = 0;
    size_t header_end_pos = 0;
    
    std::cerr << "[DEBUG] recv_response: waiting for data..." << std::endl;
    
    while (true) {
        // 从 socket 接收数据到 buffer
        ssize_t n = recv(sockfd, 
                        reinterpret_cast<char*>(recv_buffer.data()),
                        recv_buffer.size() - 1,
                        0);
        
        if (n <= 0) {
            if (n < 0) {
                std::cerr << "[ERROR] recv() failed: " << strerror(errno) << std::endl;
                response.error_message = "recv() failed: " + std::string(strerror(errno));
            } else {
                std::cerr << "[DEBUG] Connection closed by server" << std::endl;
            }
            break;
        }
        
        // 将接收的数据追加到 header_str
        header_str.append(reinterpret_cast<const char*>(recv_buffer.data()), n);
        
        // 检查是否收到完整头部
        if (!headers_complete) {
            size_t header_end = header_str.find("\r\n\r\n");
            if (header_end != std::string::npos) {
                // 找到完整头部，解析它
                std::string headers = header_str.substr(0, header_end);
                
                // 解析状态码（从第一行）
                size_t status_pos = headers.find("HTTP/");
                if (status_pos != std::string::npos) {
                    size_t status_end = headers.find("\r\n", status_pos);
                    std::string status_line = headers.substr(status_pos, status_end - status_pos);
                    
                    // 状态行格式：HTTP/1.1 200 OK
                    size_t code_pos = status_line.find(" ");
                    if (code_pos != std::string::npos) {
                        std::string code_str = status_line.substr(code_pos + 1, 3);
                        response.status_code = std::stoi(code_str);
                    }
                }
                
                // 解析 Content-Length
                size_t cl_pos = headers.find("Content-Length:");
                if (cl_pos != std::string::npos) {
                    size_t cl_end = headers.find("\r\n", cl_pos);
                    std::string cl_str = headers.substr(cl_pos + 15, cl_end - cl_pos - 15);
                    cl_str.erase(0, cl_str.find_first_not_of(" \t"));  // 去除前导空格
                    cl_str.erase(cl_str.find_last_not_of(" \t") + 1);   // 去除尾随空格
                    content_length = std::stoul(cl_str);
                }
                
                headers_complete = true;
                header_end_pos = header_end + 4;  // 跳过 \r\n\r\n
                
                // 分配 body 空间
                response.body.reserve(content_length > 0 ? content_length : 8192);
                
                // 处理头部之后的数据（body 第一部分）
                if (header_end_pos < header_str.size()) {
                    for (size_t i = header_end_pos; i < header_str.size(); i++) {
                        response.body.push_back(header_str[i]);
                    }
                    body_bytes_received = header_str.size() - header_end_pos;
                }
                
                // 检查是否已经收到完整 body
                if (content_length > 0 && body_bytes_received >= content_length) {
                    break;
                }
                
                // ✅ 清空 header_str，释放内存（不再需要）
                header_str.clear();
            }
        } else {
            // 继续接收 body 数据
            for (ssize_t i = 0; i < n; i++) {
                response.body.push_back(recv_buffer[i]);
            }
            body_bytes_received += n;
            
            if (content_length > 0 && body_bytes_received >= content_length) {
                break;
            }
        }
    }
    
    // 判断是否成功（2xx 状态码）
    response.success = (response.status_code >= 200 && response.status_code < 300);
    
    if (!response.success && response.error_message.empty()) {
        response.error_message = "HTTP request failed with status: " + std::to_string(response.status_code);
    }
    
    std::cerr << "[DEBUG] recv_response: status=" << response.status_code 
              << ", body_size=" << response.body.size() 
              << ", content_length=" << content_length << std::endl;
    
    return response;
}

// ============================================================================

HttpResponse HttpClient::download(const std::string& url, uint64_t offset, uint64_t length) {
    std::cerr << "[DEBUG] HttpClient::download: url=" << url << std::endl;
    
    std::string host = extract_host(url);
    std::string path = extract_path(url);
    uint16_t port = extract_port(url);
    
    if (use_tls(url)) {
        return get(host, path, port, "");
    }
    
    int sockfd = create_socket(host, port);
    if (sockfd < 0) {
        HttpResponse resp;
        resp.success = false;
        resp.error_message = "Failed to create socket";
        return resp;
    }
    
    Defer defer([sockfd]() {
        close(sockfd);
    });
    
    HttpRequest request("GET", host, path);
    
    if (offset > 0 || length > 0) {
        std::string range = "bytes=" + std::to_string(offset) + "-";
        if (length > 0) {
            range += std::to_string(offset + length - 1);
        }
        request.headers.push_back({"Range", range});
        std::cerr << "[DEBUG] Range request: " << range << std::endl;
    }
    
    return send_request(sockfd, request);
}

// downloadWithRetry - 带重试的下载
// ============================================================================

HttpResponse HttpClient::downloadWithRetry(
    const std::string& url,
    uint64_t offset,
    uint64_t length,
    int maxRetries,
    int timeoutSeconds
) {
    HttpResponse lastResponse;
    
    for (int attempt = 1; attempt <= maxRetries; attempt++) {
        std::cerr << "[DEBUG] Download attempt " << attempt << "/" << maxRetries << std::endl;
        
        HttpResponse response = download(url, offset, length);
        
        if (response.success) {
            std::cerr << "[DEBUG] Download succeeded on attempt " << attempt << std::endl;
            return response;
        }
        
        lastResponse = response;
        
        if (response.status_code >= 500) {
            std::cerr << "[WARN] Server error " << response.status_code 
                      << ", retrying in " << attempt << " seconds..." << std::endl;
            std::this_thread::sleep_for(std::chrono::seconds(attempt));
        }
        else if (response.status_code == 429) {
            std::cerr << "[WARN] Rate limited (429), waiting 5 seconds..." << std::endl;
            std::this_thread::sleep_for(std::chrono::seconds(5));
        }
        else if (response.status_code == 0 && 
                 response.error_message.find("timeout") != std::string::npos) {
            std::cerr << "[WARN] Connection timeout, retrying..." << std::endl;
            std::this_thread::sleep_for(std::chrono::seconds(1));
        }
        else {
            std::cerr << "[ERROR] Non-retryable error: " << response.error_message << std::endl;
            if (response.status_code > 0) {
                std::cerr << "[ERROR] HTTP status: " << response.status_code << std::endl;
            }
            break;
        }
    }
    
    std::cerr << "[ERROR] All " << maxRetries << " attempts failed" << std::endl;
    return lastResponse;
}
```
